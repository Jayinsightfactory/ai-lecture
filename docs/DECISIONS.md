- 모바일에서 PC 슬라이드를 기계적으로 좌우 분할하지 않는다.

- 문구: `@MOYI`
- PC: 중앙 대각선 반투명 워터마크
- 모바일: 콘텐츠 밖 하단 우측의 작은 `@MOYI · MOBILE EDITION`

## 콘텐츠

- 사용자가 요구하지 않은 통계와 차트를 임의로 추가하지 않는다.
- 업무 자동화 UI의 중심은 업로드, 처리 단계, 매칭, 검증, 미리보기, Excel·PDF 다운로드, 작업 이력과 로그다.
- 루프와 오케스트레이션을 구분한다.
- 위험한 외부 행동에는 사람 승인 단계를 둔다.
- 기본 청중은 사내 비개발자이며 쉬운 한국어 행동명을 우선한다.
- 새 요청은 기존 내용과 비교해 추가·대체·통합 중 하나로 처리한다.

## 원본과 생성 방식

- `slides.html`을 공개 강의의 단일 원본으로 취급한다.
- PDF는 직접 수정하지 않고 원본에서 다시 생성한다.
- PDF 생성 도구는 재현 가능한 실행 방법이 확인된 뒤 저장소 추적 여부를 결정한다.
- 기존 공개 파일명과 URL은 별도 변경 지시가 없으면 유지한다.
