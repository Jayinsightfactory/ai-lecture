# AI 업무자동화 필수 학습 자료

AI 활용과 업무 자동화 바이브 코딩에 필요한 핵심 정보를 찾아보고 바로 적용할 수 있도록 구성한 공개 학습 저장소다.

## 공개 결과물

- 필수 학습 슬라이드: `slides.html`
- PC·모바일 다운로드 선택: `downloads/index.html`
- 공개 주소: <https://jayinsightfactory.github.io/ai-lecture/>

## 작업을 시작하는 순서

1. `AGENTS.md`에서 원본, 산출물, 모바일·Git 규칙을 확인한다.
2. `docs/DECISIONS.md`에서 이미 확정된 결정을 확인한다.
3. `docs/CHANGE_CHECKLIST.md`로 변경 범위와 완료 조건을 정한다.
4. 과거 반복 문제는 `docs/GIT_HISTORY_RETROSPECTIVE.md`에서 확인한다.
5. 요청 작성은 `docs/AI_작업요청_및_재작업방지_가이드.md`의 빈칸 템플릿을 사용한다.
6. 컨텍스트 압축이나 새 세션 전에는 `docs/CURRENT_TASK_TEMPLATE.md`로 현재 상태를 인수인계한다.

## 원본과 산출물

`slides.html`이 공개 슬라이드의 원본이다. `downloads/`의 PDF는 원본에서 생성하는 산출물이며 직접 편집하지 않는다. 콘텐츠가 바뀌면 PC PDF, 모바일 PDF, 선택 페이지의 안내와 페이지 수도 함께 확인한다.

## 주의

현재 PDF 생성에 사용한 로컬 도구와 임시 파일은 저장소에 정식 빌드 도구로 확정되지 않았다. 생성 방식을 재현하려면 도구의 입력·출력·의존성·실행 명령을 먼저 문서화하고 검증한 뒤 추적한다.
